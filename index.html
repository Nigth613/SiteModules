Command Maneger

Descrição: Este script Lua foi projetado para ajudá-lo a criar comandos no roblox studio de forma mais facil.

Como Usar:

  1. Baixe o Script: Baixar o module
  2. Onde Colocar:

    Salve o arquivo `CommandModule.lua` em qualquer pasta em seu computador. Para facilitar, você pode colocá-lo em uma pasta na area de trabalho.

  3. Configuração (dentro do Script):

    Crie um Script no ServerScriptService para criar os comandos,você pode usar o 'Create' para criar os comandos e oque eles fazem, e o 'Call' para chamar os comandos

    
-- Configuração
local PREFIX = "/" -- Você pode mudar para "!" ou ":"
local ADMIN_NAMES = { "kigbloxyt" }

    Altere `PREFIX` para mudar oque vai chamar o comando exemplo '/Kick', Agora se eu mudar o `PREFIX` para ':' vai ficar assim ':Kick', mude o `ADMIN_NAMES` para seu nick no roblox

Código do Script (completo para referência):


-- Serviços
local Players = game:GetService("Players")

-- Módulo
local CommandManager = require(game.ReplicatedStorage.CommandManeger)

-- Configuração
local PREFIX = "/" -- Você pode mudar para "!" ou ":"
local ADMIN_NAMES = { "kigbloxyt" }

-- Utilitário para verificar se o jogador é admin
local function isAdmin(player)
	for _, name in ipairs(ADMIN_NAMES) do
		if player.Name == name then
			return true
		end
	end
	return false
end

-- Comando de Kick (exemplo funcional)
CommandManager:Create("kick", 2)(function(targetName, reason)
	local target = Players:FindFirstChild(targetName)
	if target then
		target:Kick(reason or "Você foi expulso.")
	else
		warn("Jogador não encontrado:", targetName)
	end
end)

-- Escutar o jogador que é admin
Players.PlayerAdded:Connect(function(player)
	if isAdmin(player) then
		player.Chatted:Connect(function(msg)
			-- Verifica se a mensagem começa com o prefixo
			if msg:sub(1, #PREFIX) == PREFIX then
				local content = msg:sub(#PREFIX + 1) -- remove o prefixo
				local args = string.split(content, " ")

				local commandName = args[1]:lower()
				table.remove(args, 1) -- remove o nome do comando dos args

				local success, err = pcall(function()
					CommandManager:Call(commandName, table.unpack(args))
				end)

				if not success then
					warn("Erro ao executar comando:", err)
				end
			end
		end)
	end
end)
